# 全国大学生电子设计竞赛_仪器/信号类 备赛参考

1.从不断复刻和抄开始，但要把自己当作电路和程序的设计者，多问AI为什么，而不是让AI直接给答案
2.技术问题AI(2026年)推荐：DeepSeek用来问知识(比较详细比较稳)，豆包用来查资料(它联网查比较新)。如果能上外网，Gemini用来问知识，Grok用来查资料(用英文提问，翻资料的神)
3.学C语言：B站搜：浙大 翁恺，多练习。初学建议Visual Studio较新版本作为开发环境
4.学STM32: 洋桃电子(F4 HAL)、阿熊话太多、keysking、安富菜(解决高难度问题)、正点原子、不要跟江科大(标准库淘汰了)
5.学画板：洋桃电子 凡亿电子 Altium中国。先学嘉立创，再转AD
6.电路仿真：LTspice、TINA。
7.焊接一定要学，最好风枪也要熟练。
8.多看立创开源平台和Github，找些有意思的项目复刻或者在它的基础上改进。
9.学有余力：Python(Pycharm)、MATLAB、树莓派(Linux+OpenCV)
10.其他B站博主推荐：JT硬件乐趣、林亦LYi、张彦欣单片机、工科男孙老师、爱上半导体、稚晖君、猫叔小栈、硬汉嵌入式
学自己热爱的，遵循行业规律的。多打板 多写程序 少焦虑。

## 基本建议<a id="section1"></a>
* 从不断复刻和抄开始，但要把自己当作电路和程序的设计者，多问AI为什么，而不是让AI直接给答案
* 技术问题AI(2026年)推荐：DeepSeek用来问知识(比较详细比较稳)，豆包用来查资料(它联网查比较新)。如果能上外网，Gemini用来问知识，Grok用来查资料(用英文提问，翻资料的神)
* 学C语言：B站搜：浙大 翁恺，多练习。初学建议Visual Studio较新版本作为开发环境
* 学STM32: 洋桃电子(F4 HAL)、阿熊话太多、keysking、安富菜(解决高难度问题)、正点原子、不要跟江科大(标准库淘汰了)
* 学画板：洋桃电子 凡亿电子 Altium中国。先学嘉立创，再转AD
* 电路仿真：LTspice、TINA。
* 焊接一定要学，最好风枪也要熟练。
* 多看立创开源平台和Github，找些有意思的项目复刻或者在它的基础上改进。
* 学有余力：Python(Pycharm)、MATLAB、树莓派(Linux+OpenCV)
* 其他B站博主推荐：JT硬件乐趣、林亦LYi、张彦欣单片机、工科男孙老师、爱上半导体、稚晖君、猫叔小栈、硬汉嵌入式
### 学自己热爱的，遵循行业规律的。多打板 多写程序 少焦虑。

## 电赛常用物料采购指南<a id="section2"></a>
图文版(暂时还没把图弄过来)：https://www.kdocs.cn/l/ckrKR1wrUbmj
各位采购元件，要始终给自己留出余量。避免出现"采购—焊接失误或烧坏了—再采购—再损坏"这样极其耽误时间的情况。
一般几毛钱的原件可以买2N的量；几块钱的原件可以买N+2；除非特别贵的芯片，不然最少多买1颗。剩下的用盒子收好，作为自己的库存。
越是新手，越要多买些备件。为自己尝试性的实验或者调试损耗做足准备。
电子元件其实不贵，做一个便宜的项目也就一两杯奶茶钱。贵的元件可以尝试找老师申请或者加入学院统一采购的清单。投资自己总是值得的。

元件采购推荐：淘宝—深圳优信电子科技有限公司、立创商城https://www.szlcsc.com/。一般这两家都能满足，质量也有保障。买原件切记看清楚封装、耐压等关键参数。


1. 电阻：建议统一购买贴片电阻本(下左图)，2-3人合用一本，一般新手用0805，老手用0603。你们会发现10K、100K、4.7K这种最常见的值会迅速用完，可以单独买一些补充。注意一般价格都是50或100个，而不是一个一个卖的。做模拟电路用的电阻注意精度要求和温漂。
2. 电容：建议统一购买贴片电容本，2-3人合用一本，一般新手用0805，老手用0603。100nF和1uF作为去耦电容最常见的值，消耗量特别大，建议买几百个100nF 50V X7R 0603和1uF 50V X7R 0805的，大家一起用。注意一般价格都是20或50个。注意耐压至少要实际工作电压的2~3倍。
3. 电解电容：以上说的都是陶瓷电容MLCC一般容量较小，针对高频干扰滤波。需要容量大的时候一般用电解电容(黑皮圆柱状)。按需求参数选择即可。注意耐压至少要实际工作电压的2倍。在关键时刻或关键位置，推荐使用同参数的固态电容替代(上右图)，它的性能非常好，价格较高。
4. 电感：一般大家的需求都是功率电感，千万不要买错了，一般是灰色小方块形状。注意电感的额定电流一定要大于你电路所需的最大电流。功率电感里也有种类区别，我只推荐一体成型电感(合金电感)(左图)或T-Core(推荐品牌：长江微电)。做电源类的同学可能会用到高压、大电流的电感(如右图)，甚至需要手工绕制，具体可以在群里问谭冲学长、王恩德学长。
5. 上面讲完了狭义上的电感，在电路中起能量存储和释放的作用，一般用于开关电源(SMPS)，这条也属于电感大类：共模滤波器、差模滤波器
6. 二极管：注意封装、耐压、额定电流即可。一般优先使用肖特基二极管，常用型号1N4148(贴片的上面会印T4)；少数使用传统的整流二极管，常用型号1N4007(贴片的上面会印M7或A7)。焊接是千万注意方向，有竖线或黑色标记一段为负极。
7. 三极管
8. MOS管

## 常用调试器和线材<a id="section3"></a>
调试器和线材，求求大家别省钱买便宜的。大多数时候调试不出来就很烦，如果最后发现是因为调试器或者线材导致的，心态会爆炸。而且很难排查到这一点。
另外，注意保护自己的电脑。如果你的电路里有大电流或者大于5V的信号，且没有做隔离，建议接USB隔离器，以免电路板上的电源窜入电脑，导致烧毁USB甚至主板和CPU。最好不要使用下载器给电路板供电，只接好地线即可。下载器的电源噪声大、质量差、电流小、保护弱。
复杂的电路都是分段排查的，要按信号或者电流的走向，分部分焊接、调试。排查时要确保前一段是没问题的，再往下排查。很多人发现后面有问题还要退回去怀疑前面电路是不是还没做好，我真的很无语。以下是常用调试器：
1. USB转串口(TTL或者说UART 连接STM32的串口和电脑的串口助手用)：
购买：https://lckfb.com/project/detail/lckfb-ch343p-usb-to-ttl-type-a?param=baseInfo
其电路介绍：https://oshwhub.com/li-chuang-kai-fa-ban/lckfb-ch343p-usb-to-ttl-type-a
J-LINK和ST-LINK都可以用来调试STM32。J-LINK还可以调试其他ARM MCU，适用性更广，功能更多。
2. J-LINK：https://item.taobao.com/item.htm?id=895384184941&skuId=5744092880255
3. ST-LINK：https://item.taobao.com/item.htm?id=724576584268&skuId=5031505112321
4. USB隔离器：https://item.taobao.com/item.htm?id=833924458949&skuId=5752530287691
5. 线材：硅胶 双绞 屏蔽 特氟龙
6. USB HUB（不一定选这款 丰俭由人）：https://detail.tmall.com/item.htm?abbucket=4&id=572026050803&skuId=6113328186791
7. 其他USB线：USB-A转Type-C、USB-A转Type-B(2.0)、USB-A公转母(延长线)、USB-A转MicroUSB(2.0)、USB-A转miniUSB(2.0)。长度一般不短于30cm，不大于1.5m是合适的。
